## Tambahkan Halaman Terproteksi

Untuk mengaktifkan proteksi password, masukkan path halaman ke object `protectedRoutes`.

```tsx
const protectedRoutes = {
  '/work/once-ui': true
};
```

Contoh di atas membuat halaman `/work/once-ui` hanya dapat diakses setelah pengguna memasukkan password.

## Atur Password

Password diatur lewat `.env.local` (lokal) atau environment variable saat deploy (misalnya di Vercel).
