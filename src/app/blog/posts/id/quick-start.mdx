## Tentang

Magic Portfolio adalah template portofolio yang lengkap, responsif, dan ramah SEO. Kontennya berbasis MDX sehingga mudah ditulis serta dikelola.

## Lisensi

Magic Portfolio menggunakan lisensi [CC BY-NC 4.0](https://creativecommons.org/licenses/by-nc/4.0/). Artinya hanya untuk penggunaan personal dan wajib mencantumkan atribusi karya asli.

Atribusi default sudah ada di footer, tetapi kamu boleh memindahkannya ke bagian lain selama tetap terlihat.

Kalau ingin memperluas lisensi untuk kebutuhan komersial, cek [Once UI Pro plan](https://once-ui.com/pricing).

## Langkah Cepat

Clone repository:

```bash
git clone https://github.com/once-ui-system/magic-portfolio.git
```

Install dependency:

```bash
npm install
```

Jalankan server development:

```bash
npm run dev
```
