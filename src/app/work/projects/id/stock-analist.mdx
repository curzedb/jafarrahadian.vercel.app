## Gambaran Umum

Stock Analist adalah platform analisis dan prediksi saham berbasis AI yang membantu user melakukan training, evaluasi, serta monitoring model dalam satu aplikasi terintegrasi. Sistem memakai pendekatan microservices: antarmuka utama di Django dan layanan pemrosesan data di FastAPI, dengan orkestrasi training asynchronous memakai Celery + Redis.

## Fitur Utama

- Arsitektur microservices (Django + FastAPI).
- Pipeline deep learning multi-model (LSTM, GRU, CNN).
- Monitoring training real-time.
- Perhitungan indikator teknikal (RSI, MACD, Bollinger Bands, MA).
- Dashboard performa model (RMSE, MAPE, RÂ²).
- Backtesting pada data historis.
- Transfer learning untuk adaptasi ticker baru.
- Explainable AI (XAI) untuk interpretasi prediksi.

## Teknologi

Python 3.10+, Django 4.x, FastAPI, Celery + Redis, TensorFlow/Keras, Scikit-Learn, SQLite/PostgreSQL, Docker, dan Docker Compose.

## Tantangan & Pembelajaran

Tantangan utama adalah sinkronisasi proses training ML yang panjang dengan responsivitas web app, termasuk eksperimen multi-model. Saya juga belajar membangun struktur service/repository agar lifecycle model (train, monitor, evaluate, explain) tetap maintainable saat fitur bertambah.

## Hasil

Proyek berkembang dari aplikasi prediksi sederhana menjadi platform ML yang lebih matang dengan arsitektur mirip production. Kini sistem mendukung workflow lengkap dari training hingga interpretasi model sebagai fondasi pengembangan AI finansial berikutnya.
