## Gambaran Umum

RFQ Automation adalah aplikasi web untuk mendigitalisasi proses Request for Quotation di PT. Astra Visteon Indonesia. Sistem mencakup alur lengkap dari pengajuan, review purchasing, revisi, komunikasi vendor, negosiasi, hingga approval akhir.

## Fitur Utama

- Dashboard interaktif status RFQ real-time.
- Role akses: **User** (requester) dan **Admin** (purchasing).
- Workflow RFQ end-to-end.
- Komunikasi negosiasi terintegrasi.
- Filter dan pencarian lanjutan.
- Fitur repeat order.
- Export data RFQ ke Excel.

## Teknologi

- Laravel (PHP) sebagai backend framework.
- SQL Server sebagai database utama.
- Struktur DB ter-normalisasi.
- Controller terpisah untuk flow user dan admin.

## Tantangan & Pembelajaran

Tantangan utama adalah memetakan workflow approval/revisi yang kompleks ke sistem digital dan memastikan adopsi pengguna berjalan lancar. Proyek ini memperkuat pengalaman pada desain skema data enterprise dan implementasi Laravel skala produksi.

## Hasil

Sistem berhasil mempercepat proses RFQ, mengurangi pekerjaan manual, dan meningkatkan transparansi lintas stakeholder melalui dashboard serta audit trail yang jelas.
