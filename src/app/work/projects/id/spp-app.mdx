## Gambaran Umum

Proyek ini dikembangkan untuk membantu analisis pasar dan prediksi harga penutupan saham secara lebih sistematis. Sistem mendukung beberapa ticker dan menguji tiga model neural network sekaligus, dengan praktik MLOps untuk otomatisasi.

## Fitur Utama

- Ingest data dinamis dari `yfinance` + cache SQLite.
- EDA interaktif berbasis Streamlit.
- Perbandingan model LSTM, CNN, dan GRU.
- Hyperparameter tuning otomatis dengan KerasTuner.
- Evaluasi metrik RMSE, MAPE, dan RÂ².
- Retraining otomatis mingguan via GitHub Actions.
- Prediksi hingga 7 hari ke depan.

## Teknologi

Python, Streamlit, TensorFlow/Keras, KerasTuner, Pandas, NumPy, SQLite, yfinance, dan GitHub Actions.

## Tantangan & Pembelajaran

Tantangan terbesar adalah konsistensi data API, tuning model multi-arsitektur, dan orkestrasi retraining otomatis. Proyek ini memperkuat pemahaman mengenai bridging model development ke deployment workflow.

## Hasil

Aplikasi berhasil memberikan platform prediksi yang otomatis dan user-friendly. Model GRU mencapai performa tinggi dengan MAPE sekitar 0.99%, serta fondasi MLOps untuk update model berkala.
