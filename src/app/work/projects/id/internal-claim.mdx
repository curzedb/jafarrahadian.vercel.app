## Gambaran Umum

Internal Claim Web Application dikembangkan untuk mendigitalisasi proses penanganan masalah produksi di PT. Astra Visteon Indonesia. Tujuannya meningkatkan kecepatan respons, akuntabilitas antar departemen, serta sentralisasi data untuk analitik dan pelaporan.

## Fitur Utama

- **Dashboard Statistik Dinamis** untuk status issue dan beban kerja.
- **Akses Multi-Role**: Production, Quality, Engineering, R&D, dan Admin.
- **Workflow Eskalasi Bertahap** dari laporan awal hingga penyelesaian lintas divisi.
- **Notifikasi Email Otomatis** pada setiap tahapan penting.
- **Draft & Final Submission** untuk analisa sebelum submit final.
- **Pelaporan Terpusat + Export Excel**.
- **Admin Panel** untuk user, role, dan data master.

## Teknologi

- Laravel (PHP) dengan arsitektur MVC.
- SQL Server sebagai database utama.
- RBAC untuk kontrol akses.
- AJAX filtering untuk pengalaman dashboard yang responsif.

## Tantangan & Pembelajaran

Tantangan utama adalah menerjemahkan logika eskalasi lintas departemen ke software yang konsisten dan minim bug. Implementasi RBAC granular juga menjadi pengalaman penting untuk aplikasi enterprise.

## Hasil

Aplikasi berhasil mendigitalisasi alur internal claim secara end-to-end, meningkatkan efisiensi operasional hingga target ~40%, sekaligus memperkuat transparansi dan akuntabilitas proses.
